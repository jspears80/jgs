# JGS Site — Stable v2 (Sanitized)

This is a portable React + Vite + Tailwind project that contains the **Stable v2** page
from your canvas. It avoids Tailwind bracket utilities and TypeScript so it runs cleanly
in strict environments (GitHub, CI, etc.).

## Quick start
```bash
npm install
npm run dev
```
Open http://localhost:5173

## Build
```bash
npm run build
npm run preview
```
