# JGS Cloud Compliance — Site

## Run locally
```bash
npm i
npm run dev
```

## Deploy to Vercel
- Push to GitHub
- Import on Vercel as Next.js 14 app
- Build command: `npm run build`
- Output: Next.js default

## SEO files
- `public/robots.txt`
- `public/sitemap.xml`
- `app/layout.tsx` includes OpenGraph and icons
